## Basic Python Concepts 

1. Module 
    - Its a single file that can be imported in other Python programs.

2. Package
    - Its a collection of related modules organized in directories. 
    - package has __init__.py file to indicate that the directory is a package.

3. Library
    - A library is a collection of packages and modules that provide specific functionality.
    - Example: NumPy, Pandas, Matplotlib etc.
